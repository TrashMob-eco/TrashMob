# Quick Command: Add Project to Planning

**For AI Assistants:** Use this checklist when user says "add a new project"

---

## ? Quick Checklist

### 1. Get Info from User
- Project title
- Business need (why?)
- Priority (Low/Medium/High/Critical)
- Size (Very Small ? Very Large)
- Risk (Low/Medium/High/Very High)

### 2. Determine Project Number
```powershell
# Find next number (currently 25, so next is 26)
(Get-ChildItem Planning/Projects/Project_*.md | 
    Measure-Object).Count + 1
```

### 3. Create File: `Planning/Projects/Project_XX_Title.md`
- Copy `_Project_Template.md`
- Fill all sections
- Add code examples if technical
- Complete footer

### 4. Update README.md
- Add to navigation (Q1/Q2/Q3/Q4/TBD section)
- Add to theme quick links
- Update status table count

### 5. Update TODO List
- Add checkbox to `TODO_Project_Extraction.md`

### 6. If High Priority: Update Executive Summary
- Add to roadmap section
- Add to priority matrix

### 7. If High Risk: Update Risks Document
- Add project-specific risk
- Update risk dashboard table

### 8. Quality Check
- [ ] No `[PLACEHOLDER]` text
- [ ] All links work
- [ ] Tables format correctly
- [ ] Footer complete
- [ ] At least 3 risks listed

### 9. Tell User
```markdown
? **Project XX - [Title] created!**

**Files created:** Planning/Projects/Project_XX_Title.md
**Files updated:** README.md, TODO list[, Executive Summary, Risks]

**Next:** Review file, fill TODOs, get approval
```
